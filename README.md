# 战锤40K：虚空黎明

## ⚔️ 游戏简介

《战锤40K：虚空黎明》是一款基于战锤40K世界观的AI驱动回合制网页游戏。

玩家扮演极限战士钛-7，在荷鲁斯之乱的动荡时期，通过战斗、建设、选择来建立势力，揭开自己的命运。

## 🛠️ 本地运行

### 环境要求
- Node.js 14+
- npm 或 yarn

### 安装步骤
```bash
# 1. 进入项目目录
cd warhammer-game

# 2. 安装依赖
npm install

# 3. 启动服务器
npm start

# 4. 打开浏览器访问
# http://localhost:3000
```

## 🎯 游戏功能

### 核心系统
- **战斗系统**: 迎击敌人/防守要塞
- **建筑系统**: 升级巢穴/建造设施
- **调查系统**: 与NPC对话找出内鬼
- **探索系统**: 北境荒野/古代遗迹
- **回合系统**: 每回合3次行动

### 资源管理
- 物资: 用于建筑升级
- 灵魂点: 特殊资源
- 声望: 阵营关系
- 混沌值: 角色状态（0-100）

### NPC互动
- 塔姆 (接待员) - 可疑度3/10
- 卡尔 (军需官) - 可疑度5/10  
- 尤里 (机械教神甫) - 可疑度4/10

## 📡 API接口

游戏提供REST API：

```
GET  /api/game/:sessionId  获取游戏状态
POST /api/action          执行行动
POST /api/endTurn         回合结束
GET  /api/log/:sessionId  获取游戏日志
```

## 🎨 技术栈

- **前端**: HTML5, CSS3, JavaScript (ES6+)
- **后端**: Node.js, Express.js
- **存储**: 内存存储（生产环境可扩展为数据库）
